# Task 1 - Landing Page

## Description
This project is a responsive and visually appealing Landing Page created using HTML and CSS. It demonstrates a clean layout, attractive design, and user-friendly interface.

## Features
- Responsive design
- Navigation bar
- Hero section
- Call-to-action buttons
- Clean and modern layout

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone the repository.
2. Open the Task1_LandingPage folder.
3. Open `index.html` in any web browser.

# Task 2 - Personal Portfolio

## Description
This project is a personal portfolio website developed using HTML, CSS, and JavaScript. It showcases a professional layout with sections such as About, Skills, Projects, and Contact.

## Features
- Responsive portfolio design
- About section
- Skills section
- Projects section
- Contact section
- Smooth navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open the Task2_Portfolio folder.
3. Double-click the index.html file or open it in any web browser.

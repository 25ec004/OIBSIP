# Task 3 - Temperature Converter

## Description
A simple and interactive Temperature Converter web application that converts temperatures between Celsius and Fahrenheit. It validates user input and displays the converted temperature with the correct unit.

## Features
- Celsius to Fahrenheit conversion
- Fahrenheit to Celsius conversion
- Input validation
- Responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open the Task3_TemperatureConverter folder.
3. Double-click the index.html file or open it in any web browser.
